SIG Berbasis WEB

Teknik Geodesi UGM

Sistem Informasi Geografis (SIG) merupakan sistem berbasis komputer yang digunakan untuk memanipulasi, menganalisis, menyimpan, dan menganalisis informasi geografis. SIG berbasis web adalah SIG yang diakses melalui internet dan dapat diakses dari berbagai perangkat seperti komputer, tablet, atau smartphone. SIG berbasis web terdiri dari beberapa komponen seperti data spasial, data atribut, perangkat keras, perangkat lunak, dan pengguna. SIG berbasis web dapat digunakan untuk melakukan berbagai analisis seperti analisis klasifikasi, analisis overlay, analisis networking, analisis buffering, dan analisis lainnya. Penggunaan SIG berbasis web memiliki beberapa keuntungan seperti pengumpulan data yang hemat tempat, mudah dalam pengolahan data, dan dapat diakses dari berbagai perangkat.

Bahasa HTML, CSS, dan Java Script

Ketiga bahasa tersebut adalah bahasa internet. HTML (HyperText Markup Language) biasanya diibaratkan sebagai objek, CSS diibaratkan sebagai style, dan JavaSript diibaratkan sebagai fungsi yang menggerakan. HTML, CSS, dan JS ini memiliki struktur yang berbeda-beda.

Client-Side VS Server Side

Dalam pemrograman, terdapat dua buah cara untuk memproses request dari pengguna, yaitu client-side dan server-side. Client-Side adalah ketika script dimasak pada Browser. Sedangkan, server-side adalah ketika script dimasak oleh server sebelum disajikan ke Brwoser.

Framework dan Library

Framework dan libraries adalah aplikasi fungsional unik dan dapat dimanfaatkan untuk orang lain. Cakupan libraries lebih kecil daripada framework. Salah satu contoh dari libraries adalah JavaScript. Beberapa contoh framework dan libraries yang dapat digunakan yaitu Bootstrap dan LeafletJS.
